# BNMR — Bayesian Network Meta-Learning (Liu et al., FAccT 2025)

> **First work** em **face component fairness** (granularidade
> biológica). Propõe BNMR — Bayesian Network-informed Meta Reweighting.
> Sugere que face component fairness pode ser **surrogate objective**
> para demographic fairness. **Baseline competitivo recente** para
> nosso Cap 2.

## 1. Resumo do problema atacado

> *Fonte: abstract verbatim.*

Trabalhos prévios em FR fairness focaram em demographic fairness
(race, gender). **Fairness ao nível de componentes biológicos da face**
permanece inexplorada.

Dois desafios identificados:

1. **Attribute label scarcity** — labels para componentes
   biológicos são raros.
2. **Attribute inter-dependencies** — componentes não são
   independentes.

## 2. Método

> *Fonte: abstract verbatim.*

**BNMR — Bayesian Network-informed Meta Reweighting**:

- **Bayesian Network calibrator** rastreia bias do modelo em tempo
  real durante treino.
- Encoda **prior probabilities** para atributos de componentes
  faciais.
- Guia **adaptive meta-learning-based sample reweighting**.
- Endereça os 2 desafios (scarcity + dependencies).

> **[PENDENTE PDF]** Estrutura exata da Bayesian Network.
> Algoritmo de meta-learning. Como prior probabilities são definidas.
> Datasets exatos (abstract diz "large-scale real-world face dataset"
> sem nomear).

## 7. Limitações que identifiquei (a partir do abstract apenas)

- **Pressuposto forte** que o usuário define corretamente a estrutura
  causal entre atributos faciais.
- **Custo computacional adicional** da rede bayesiana.
- **Dataset não especificado no abstract** — pode ser CelebA (mais
  comum em facial attribute) — não FairFace race 7-class.
- **First work em face component fairness** — generalização para
  race 7-class do FairFace não testada.

## 8. Relação com nossa pesquisa

### 8.1 Baseline competitivo recente

BNMR é **mais recente** que FSCL+ (2022), Group DRO (2020),
Adversarial debiasing (2018). Como tese argumenta comparar contra
SOTA atual, BNMR é candidato a baseline forte.

### 8.2 Mecanismo ortogonal ao nosso

- BNMR: sample reweighting + Bayesian prior.
- Nossa abordagem: FiLM-conditioning com vetor MST.

**Não competem diretamente** — atacam fairness em camadas diferentes
(reweighting vs conditioning). Combinação possível mas não testada.

### 8.3 Insight sobre "surrogate objective"

BNMR sugere que **face component fairness** (atributos biológicos)
pode ser surrogate para demographic fairness. Conceitualmente
relacionado à nossa estratégia: **skin tone (componente fenotípico)
como surrogate de race fairness**.

## 9. Pontos para citar

  BNMR (Bayesian Network-informed Meta Reweighting) que tracking
  dinâmico de viés via rede bayesiana supera baselines recentes."*

  paradigmaticamente alinhada com a estratégia desta dissertação,

  para mitigação de viés em race classification — uma instância
  específica de 'componente fenotípico como surrogate'."*
