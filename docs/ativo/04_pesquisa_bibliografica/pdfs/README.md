## 🛠 Reprodutibilidade — como regenerar arquivos derivados

Os arquivos `.txt` (texto extraído via pypdf) são **gitignored** —
podem ser regenerados a qualquer momento:

```bash
# Reinstalar pypdf se necessário
pip install pypdf

# Re-extrair todos os PDFs
for pdf in docs/ativo/04_pesquisa_bibliografica/pdfs/*.pdf; do
    base=$(basename "$pdf" .pdf)
    python docs/ativo/04_pesquisa_bibliografica/_extract_pdf.py "$pdf" \
        2>/dev/null > "docs/ativo/04_pesquisa_bibliografica/pdfs/${base}.txt"
done
```

## 📜 Política de versionamento

- **PDFs open access:** versionados aqui para rastreabilidade
  acadêmica e reprodutibilidade. Todos têm licença compatível com
  redistribuição em contexto acadêmico (arXiv non-exclusive, NIST
  Public Domain, NeurIPS proceedings open, Nature SR CC-BY 4.0,
  PMLR open, Idiap open repository, NSF PAR).
- **PDFs paywalled:** **NÃO** versionados. Apenas link oficial nas
  fichas e neste README.
- **Em caso de DMCA takedown notice:** PDFs específicos podem ser
  removidos individualmente; links e fichas permanecem.
