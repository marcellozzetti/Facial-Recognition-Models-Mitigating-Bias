# Survey: Fairness in DL Vision & Language (Parraga et al. — ACM CSur 2025 / MALTA Lab PUCRS)

> **Survey BRASILEIRO** (PUCRS, Porto Alegre — financiado pela Motorola
> Mobility Brasil e CAPES). 40 páginas em ACM CSur (IF top). Propõe
> **taxonomia novel** específica para deep learning research em
> fairness. Cobre vision + language methods de debiasing.

## 1-2. Resumo + estrutura

- **Origem nacional**: Machine Learning Theory and Applications
  (MALTA) Lab, PUCRS — relevante para citar como produção
  acadêmica nacional alinhada.
- **Financiamento**: Motorola Mobility Brasil + CAPES (Finance
  Code 001).
- **Premissa do paper**: redes neurais são "correlation machines"
  propensas a modelar viés nos dados em vez de relações causais
  úteis — problema sério em domínios afetados por raça, gênero,
  idade.
- **Contribuição principal**: taxonomia novel para organizar a
  literatura de fairness em deep learning vision + language,
  construindo sobre propostas anteriores mas **tailored para
  pesquisa em DL**.
- **Conteúdo**:
  - Revisão dos principais métodos neural-based para fairness
  - Métricas de avaliação
  - Desafios atuais, trends e direções futuras
