# Bias as Subspace (Nov 2025)

> Reformulação geométrica: **bias é subespaço**, não coordenada
> única. Post-hoc debiasing deve trabalhar em níveis subspaciais.

## 1-2. Resumo

- **Repensar geometricamente** o que é viés em VLMs.
- **Bias = subespaço** no espaço de features.
- Post-hoc debiasing deve **projetar para complement** do subespaço
  de viés.

  para o complemento ortogonal desse subespaço."*
