# Towards Counterfactual Fairness (ICLR 2025)

> **Counterfactual fairness** via causal mechanisms para modelar
> discrimination ao nível individual. Predições consistentes mesmo
> se sensitive attribute fosse trocado.

## 1-2. Resumo

- **Counterfactual fairness**: predição igual no actual world e
  counterfactual onde indivíduo pertenceria a grupo diferente.
- Causal mechanisms para modelar discrimination.

## 7. Aplicação ao pipeline v3.2

- **Paradigma alternativo** ao nosso (group fairness via DR).
- Citado como **direção futura** mas não testado nesta dissertação.
- Combina com Pearl, Kusner (mencionado em Mehrabi 2021).

## 8. Citar

- *"Counterfactual fairness via causal mechanisms (ICLR 2025)
  oferece paradigma alternativo a group fairness, exigindo que
  predições sejam consistentes mesmo se o atributo sensível do
  indivíduo fosse contrafactualmente trocado. A presente
  dissertação opera com group fairness (Disparity Ratio +
  worst-class F1) por compatibilidade com a literatura de
  race classification, registrando counterfactual fairness como
  direção complementar."*
